# Building a Telegram Bot with Node.js

## Installation

cp .env.sample to .env

```nodejs 
    $ npm install
```
```nodejs
    $ nodemon index.js
```


## Make endpoint sever nodejs with [ngrok](https://dashboard.ngrok.com/cloud-edge/endpoints) temporary 

```Run start endpont with ngrok
    $ ngrok http 5000
```
